# AWF's Members

Application de gestion des cotisations et contributions.

## Installation locale

```bash
npm install
npm run dev
```

## Déploiement sur Vercel

1. Pusher ce dossier sur GitHub
2. Connecter le repo sur vercel.com
3. Déployer automatiquement

## Technologies
- React + Vite
- Supabase (base de données + authentification)
- Vercel (hébergement)
